女巫（Witch）

概述

 

女巫是指一种拥有源自异界的强大魔法力量的法师。法师通常在专门的法术学院或者博学的导师那里学习基本的施法知识，而女巫，则从其他位面的强大存在和它们的仆从，亦或其他女巫那里学习魔法技艺。

 

职业关系

 

女巫角色的同伴不需要对她有恐惧和偏见，特别是当她在生死关头证明过自己之后。然而，在女巫和她的同伴之间始终隔着一层猜疑的面纱，因为他们无法让自己完全信任她。任何拥有超自然力量的玩家角色，特别是那些拥有原始和纯朴背景的pc，从不会亲近一个女巫，而且会极力避免与她单独相处。某些时候甚至会指责非难她背叛队伍或者给他们带来厄运。

扫帚飞行

 

 

6级时，女巫可以将一把普通的扫帚施以魔法并骑着飞行，类似对扫帚使用“飞行术”。

飞行速度60ft良好，如果穿着中/重型盔甲，或者中/重度负荷，速度降为40ft。

使用该能力需要消耗一个法术位，飞行持续时间为 1分钟x消耗的法术位。
